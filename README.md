# ☕ Exercícios de Lógica em Java - Semestre 01

Este repositório contém uma coleção de programas desenvolvidos em Java para a disciplina de programação do primeiro semestre na UNA. Os códigos focam em:

- Estruturas de decisão (`if/else`, `switch`)
- Laços de repetição (`while`)
- Manipulação matemática

---

## 🚀 Tecnologias Utilizadas

- **Linguagem:** Java (JDK 21)  
- **IDE:** VS Code (com Extension Pack for Java)  
- **Sistema Operacional:** CachyOS (Linux)  

---

## 📂 Descrição dos Projetos

| Arquivo                 | Funcionalidade |
|------------------------|----------------|
| **Operacoes.java**     | Calculadora completa: realiza soma, subtração, multiplicação, divisão e potência (`Math.pow`) em um loop contínuo até o usuário digitar `"sair"`. |
| **equacao.java**       | Resolução de equações de 1º e 2º grau utilizando a fórmula de Bhaskara, tratando casos de delta negativo ou coeficientes nulos. |
| **MaquinaDeVendas.java** | Simulador de troco: calcula a menor quantidade de notas (de R$100 a R$1) necessárias para o troco, validando o valor pago. |
| **circuloEsfera.java** | Geometria: menu interativo para calcular o perímetro, área ou volume (esfera) com base em um raio fornecido. |
| **mediaAritmetica.java** | Estatística básica: recebe três números e identifica automaticamente o maior, o menor e a média aritmética. |
| **NumeroRandom.java**  | Sorteio e lógica: gera um número aleatório entre dois inteiros informados e verifica se o resultado é **par** ou **ímpar**. |

---

## 📌 Observações

Este projeto tem fins educacionais e foi desenvolvido como parte do aprendizado inicial em programação com Java.
